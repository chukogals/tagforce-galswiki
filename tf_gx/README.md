# TFGX角色概要

TFGX是以《游戏王 怪兽之决斗GX》剧情为基础设计的PSP游戏《游戏王怪兽之决斗GX TAGFORCE 1-3》的简称。共有15个来自欧贝利斯克蓝的女学生，其中5人TF2才出现。她们也全都出现在5DS的世界观中。除5位**还在上学的奇怪的孩子**以外，其他10位都成为了大人，其中有两位角色在日版中修改了名字。
